Variables

Variables in python are very easy to create and change.

Multiple assignments, constants, variables within strings are some of the ways you can use variables. Below is an example of variables within strings with an "f" statement.

  • variable = "John Doe"
  • print(f"Hello {variable}")

    First, understand you need to understand how to index into a list. The first element in a list is always index 0.

  • pizza_list = ["pepperoni", "cheese", "supreme"]

    Dictionaries are a series of key-value pairs.
